Instrucoes para compilar/rodar:
1) Copiar o vsvars32.bat da sua instalacao do visual studio para a pasta /Batches
2) Abrir o terminal na pasta /Batches
3) Rodar vsvars32.bat
4) Rodar TudoTestes.bat
5) Feche o output de texto no formato .err, contendo informações da compilação ou possíveis warnings/erros.
6) Observe os resultados dos testes no terminal.
7) Será aberto um arquivo de texto com o estado final dos contadores.
8) Se desejar jogar uma partida, execute ExecutaPartida.bat contido nessa mesma pasta.
OBS: Para visualizar o tabuleiro inteiro, recomendo rodar o terminal em fullscreen


TudoTestes.bat roda os batches Limpa.bat, Compila.bat e Executa.bat.
Limpa.bat limpa construtos de compilações anteriores.
Compila.bat gera os arquivos make e compila os módulos de testes a partir disso.
Executa.bat executa todos os testes em sequência.

Caso queira rodar eles individualmente:
1) Vá pelo terminal até a pasta /Produto
2) Execute o .exe desejado pelo terminal

TRAB4-1.exe -> Teste do módulo Lista Circular
TRAB4-2.exe -> Teste do módulo Jogador
TRAB4-3.exe -> Teste do módulo Lista Duplamente Encadeada
TRAB4-4.exe -> Teste do módulo Tabuleiro
TRAB4-5.exe -> Teste do módulo Lista Circular usando o módulo de conta do arcabouço
TRAB4-6.exe -> Partida de Ludo usando o módulo Partida
